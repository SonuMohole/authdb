# Auth Project (Minimal)

This archive contains a minimal full-stack auth project with:
- Backend: Node.js + Express + TypeScript + Prisma (PostgreSQL)
- Frontend: React + TypeScript (Vite)

.env files in backend contain example credentials. Replace them with your real credentials before running.

Quick backend steps:
1. cd backend
2. npm install
3. npx prisma generate
4. npx prisma migrate dev --name init
5. npm run dev

Quick frontend steps:
1. cd frontend
2. npm install
3. npm run dev
